8 RGB LED Module
----------------
The module consists of:
    - 8 x WS2812, each requiring only one pin to control, and supports cascade.
    - Each LED has 3 integrated LEDs (red, green, blue).
    - 256 level brightness adjustment.
    - 2^24 = 16 777 216 different colours.

Pin Description:

IN--------------------------
SYMBOL              FUNCTION
  S                  Input control signal.
  V                  Power supply pin, +3.5V-5.5V
  G                  GND

OUT-------------------------
SYMBOL              FUNCTION
  S                  Output control signal.
  V                  Power supply pin, +3.5V-5.5V
  G                  GND


SKETCH
------
- Uses a library named: Freenove_WS2812_Lib_for_ESP32".
